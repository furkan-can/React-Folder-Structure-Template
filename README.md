# React Folder Structure Template

### How to run the project

```javascript
npm i
```
and
```javascript
npm run dev
```

## Used Technologies

- Axios
- Tailwindcss
- React Icons
- React Router Dom (Latest React Router v6)
- Prop Types
- Redux-Toolkit
